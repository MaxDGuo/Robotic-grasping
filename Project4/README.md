# 基于GR-ConvNet的机器人抓取检测与机械臂抓取系统
本项目为深蓝学院《机器人抓取与操作》课程Project4实践成果，基于PyTorch实现了GR-ConvNet生成式抓取检测网络，并结合RRT路径规划、逆运动学求解，在MuJoCo仿真环境中完成了从RGB-D图像输入到机械臂无碰撞抓取的端到端任务闭环。

## 我的核心工作
1.  **GR-ConvNet网络完整实现**：基于课程框架，独立完成grconvnet.py中全部TODO开发，包括下采样卷积层、5个残差块、转置卷积上采样分支，以及抓取位置、角度、宽度的4个输出头，完成了网络前向传播全流程的开发与调试。
2.  **模型训练与性能调优**：基于Cornell Grasp抓取数据集完成模型训练，完成数据集预处理、训练流程搭建与超参数调优，50轮训练后验证集最优IoU达0.97，完成模型性能评估与单张图像离线推理可视化，分析了不同物体外形对抓取质量的影响。
3.  **端到端视觉抓取Pipeline搭建**：完成rrt_grcn_grasp.py中TODO开发，将GR-ConvNet输出的抓取位姿与机器人运动控制结合，实现了“抓取位姿检测→坐标系转换→IK逆解→RRT无碰撞路径规划→机械臂抓取执行”的完整任务闭环。
4.  **关键问题解决**：完成了转置卷积上采样的尺寸对齐、Xavier权重初始化、Dropout正则化等网络核心细节的实现；解决了仿真中抓取位姿坐标系对齐、关节限位约束等工程问题，实现了机械臂稳定抓取。

## 技术栈
- **深度学习框架**：PyTorch
- **物理仿真平台**：MuJoCo 2.3.7、dm_control
- **机器人学工具**：ikpy、transformations
- **数值计算与视觉处理**：NumPy、OpenCV、Matplotlib

## 项目成果
1.  完整复现了GR-ConvNet抓取检测网络，基于Cornell数据集完成训练与验证，50轮训练后验证集最优IoU达0.97，实现了复杂场景下抓取位姿的精准预测；
2.  搭建了从视觉感知到机器人动作执行的完整抓取Pipeline，在MuJoCo仿真环境中实现了稳定的无碰撞抓取，生成了完整的任务演示视频；
3.  输出了详细的学习笔记，记录了网络核心模块原理、代码实现细节与调试过程。

## 学习收获
掌握了基于深度学习的机器人抓取检测核心原理，理解了视觉感知算法与机器人运动规划的结合方法，熟悉了具身智能视觉抓取任务从模型训练到工程落地的完整流程。
