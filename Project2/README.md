# 基于RRT-Connect算法的PiPER机械臂路径规划与抓取系统
本项目为深蓝学院《机器人抓取与操作-经典方法与端到端》课程Project2实践成果，基于MuJoCo物理仿真平台，实现了6自由度PiPER机械臂的无碰撞路径规划与完整抓取任务，完成了从原始RRT算法到RRT-Connect双向规划算法的改进与性能优化。

## 我的核心工作
### 1. RRT-Connect双向路径规划算法实现
基于课程提供的原始RRT单树算法框架，独立重构实现了RRT-Connect双向规划算法：
- 设计起点、终点双随机树并行扩展机制，大幅降低高维关节空间的搜索难度；
- 实现树间连接逻辑，当两棵树的扩展节点距离小于阈值时完成路径拼接；
- 优化随机采样策略，保留目标点偏向采样，平衡探索性与收敛速度；
- 适配PiPER机械臂6个关节的运动限位，保证生成路径的物理可行性。

### 2. 完整抓取任务Pipeline实现
独立完成课程脚本中所有TODO模块的开发与调试，实现端到端的抓取任务闭环：
- 基于ikpy库与URDF模型，完成目标位姿到关节角度的逆运动学求解；
- 基于线性插值实现关节空间平滑轨迹生成，完成抓取前下降、抓取后上抬的平稳运动控制；
- 实现基于MuJoCo的实时碰撞检测，区分非法碰撞与夹爪合法接触；
- 实现双相机同步渲染，生成完整的任务执行演示视频。

### 3. 算法调优与问题解决
- 优化扩展步长、路径分辨率等核心参数，将窄通道场景规划成功率从90%提升至98%；
- 解决仿真调试中机械臂碰撞打飞目标物体的问题，通过调整目标位姿初始高度、下降距离，实现稳定无碰撞抓取；
- 整理了完整的学习笔记，拆解了RRT系列算法核心逻辑与调试过程。

## 技术栈
- **编程语言**：Python 3.8

- **机器人学工具**：ikpy、transformations
