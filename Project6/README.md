# 基于PPO强化学习的PiPER机械臂运动控制与抓取系统
本项目为深蓝学院《机器人抓取与操作》课程Project6实践成果，基于Stable Baselines3的PPO算法，在MuJoCo物理仿真环境中，完成了6自由度PiPER机械臂逆运动学求解、目标物体抓取两大任务的强化学习环境搭建、算法实现、模型训练与调优全流程开发。

## 我的核心工作
1.  **强化学习环境搭建与任务建模**：基于Gym接口与MuJoCo物理引擎，独立完成了两大任务的RL环境开发。针对逆运动学位姿跟踪任务，设计了14维观测空间、6维动作空间与分阶段奖励函数；针对物体抓取任务，设计了16维观测空间、碰撞检测机制与抓取成功奖励规则，实现了关节限位约束、随机目标采样、仿真状态同步等核心逻辑。
2.  **PPO算法实现与训练调优**：基于Stable Baselines3框架，设计了256+128双分支MLP策略网络结构，完成了超参数调优（episode长度、n_steps、n_epochs、总训练步数等）；解决了多进程训练与MuJoCo渲染的OpenGL资源冲突、仿真卡死等核心工程问题，完成了两个任务的端到端模型训练。
3.  **训练结果分析与问题优化**：针对训练中机械臂抖动、任务成功率低等问题，完成了奖励函数权重调优、代码逻辑重构与资源释放优化；基于TensorBoard完成了训练过程全链路监控，输出了详细的调优笔记与训练结果分析报告。

## 技术栈
- **强化学习框架**：Stable Baselines3、OpenAI Gym
- **物理仿真平台**：MuJoCo
- **机器学习框架**：PyTorch
- **数值计算与机器人学工具**：NumPy、SciPy
- **可视化工具**：TensorBoard、Matplotlib

## 项目成果
1.  **机械臂逆运动学任务**：完成9000万步模型训练，累计奖励持续正向提升，episode平均长度收敛至615步，任务最高成功率达19%，实现了6自由度机械臂从关节空间到笛卡尔空间的端到端位姿跟踪。
2.  **机械臂抓取任务**：完成960万步模型训练，模型平均奖励收敛至681，仿真帧率稳定在2289fps，任务最高成功率达25%，实现了机械臂对随机位置目标物体的端到端抓取控制。
3.  完整输出了两大任务的训练过程TensorBoard可视化曲线、详细的调优笔记与问题分析报告，形成了完整的强化学习机器人控制项目落地文档。

## 学习收获
掌握了强化学习在机器人运动控制领域的核心应用方法，熟悉了基于Gym接口的强化学习环境搭建、奖励函数设计、PPO算法的调优与工程落地流程；深入理解了机器人逆运动学、抓取任务的核心控制逻辑；具备了强化学习仿真训练中的问题定位、工程优化与结果分析能力。
