## 🎯 核心项目展示

### Project 2: 基于 RRT-Connect 的机械臂路径规划与抓取系统

**技术亮点**: 传统运动规划算法的改进与优化

- 🎯 **RRT-Connect 双向规划算法**: 独立实现双随机树并行扩展机制，路径规划成功率从 90% 提升至**98%**，收敛速度提升约**40%**
- 🤖 **完整抓取 Pipeline**: 逆运动学求解 → 关节空间轨迹生成 → 碰撞检测 → 稳定抓取
- 🔧 **工程优化**: 解决窄通道规划、碰撞打飞物体等实际问题，实现稳定无碰撞抓取
- 📊 **成果**: 100% 完成课程要求，输出完整学习笔记与仿真演示视频

**核心技术**: `RRT-Connect` `逆运动学` `MuJoCo` `路径规划`

👉 [查看详情](Project2/README.md)

---

### Project 3: 基于 PoseCNN 的 6D 物体姿态估计与抓取系统

**技术亮点**: 深度学习视觉感知与机器人控制的结合

- 🧠 **PoseCNN 网络实现**: 完成三分支网络架构 (实例分割、3D 平移估计、RoIPooling 旋转回归)
- 🔄 **端到端 Pipeline**: RGB 图像 → 6D 位姿估计 → 坐标转换 → IK 逆解 → RRT 规划 → 抓取执行
- 🎓 **深度学习实践**: 掌握从论文复现到工程落地的完整流程

**核心技术**: `PoseCNN` `6D 姿态估计` `PyTorch` `视觉伺服`

👉 [查看详情](Project3/README.md)

---

### Project 4: 基于 GR-ConvNet 的机器人抓取检测系统

**技术亮点**: 生成式抓取检测网络与大规模数据集训练

- 🕸️ **GR-ConvNet 完整实现**: 5 个残差块 + 转置卷积上采样 + 4 输出头 (位置、角度、宽度)
- 📈 **模型性能优异**: Cornell 数据集训练 50 轮，验证集最优 **IoU 达 0.97**
- 🖼️ **数据处理能力**: 完成 Cornell 数据集预处理、增强、训练全流程
- 🎯 **工程落地**: 解决转置卷积尺寸对齐、Xavier 初始化、Dropout 正则化等核心细节

**核心技术**: `GR-ConvNet` `抓取检测` `Cornell 数据集` `残差网络`

👉 [查看详情](Project4/README.md)

---

### Project 6: 基于 PPO 强化学习的机械臂运动控制系统

**技术亮点**: 强化学习在机器人控制中的应用实践

- 🎮 **RL 环境搭建**: 基于 Gym 接口设计 14/16 维观测空间、分阶段奖励函数
- 🧠 **PPO 算法实现**: 设计 256+128 双分支 MLP 策略网络，完成 9000 万步训练
- 📊 **训练成果**: 
  - 逆运动学任务：成功率**19%**，episode 长度收敛至 615 步
  - 抓取任务：成功率**25%**，平均奖励收敛至 681
- 🔧 **工程优化**: 解决多进程训练资源冲突、仿真卡死、机械臂抖动等问题

**核心技术**: `PPO` `强化学习` `Stable Baselines3` `端到端控制`

👉 [查看详情](Project6/README.md)

## 🛠️ 技术栈总览

### 编程语言
- **Python 3.8+** (主要开发语言)

### 深度学习框架
- **PyTorch** - PoseCNN、GR-ConvNet、PPO 策略网络
- **VGG16** - 特征提取 backbone
- **Stable Baselines3** - 强化学习算法库

### 机器人学与运动规划
- **RRT/RRT-Connect** - 路径规划算法
- **ikpy** - 逆运动学求解
- **transformations** - 四元数、旋转矩阵变换

### 物理仿真
- **MuJoCo 2.3.7** - 物理引擎
- **dm_control** - MuJoCo Python 接口

### 数据处理与可视化
- **NumPy** - 数值计算
- **OpenCV** - 图像处理
- **Matplotlib** - 可视化
- **TensorBoard** - 训练监控

### 数据集
- **Cornell Grasp Dataset** - 抓取检测数据集
