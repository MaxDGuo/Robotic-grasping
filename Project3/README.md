# 基于PoseCNN的6D物体姿态估计与机械臂抓取系统
本项目为深蓝学院《机器人抓取与操作》课程Project3实践成果，基于PyTorch实现了PoseCNN 6D姿态估计网络，并结合逆运动学IK、RRT路径规划，在MuJoCo仿真中完成了从RGB图像到机械臂无碰撞抓取的端到端任务。

## 我的核心工作
1.  **PoseCNN网络核心模块实现**：基于课程框架，独立完成了pose_cnn.py中全部TODO开发，包括实例分割分支、3D平移估计分支、基于RoIPooling的3D旋转回归分支，跑通了网络训练与推理全流程。
2.  **端到端视觉抓取Pipeline搭建**：完成rrt_piper_grasp.py中TODO开发，将PoseCNN输出的6D位姿与机械臂控制结合，实现了“位姿估计→坐标系转换→IK逆解→RRT路径规划→抓取执行”的完整闭环。
3.  **关键问题解决**：严格遵循论文细节，规避了平移/旋转分支错误添加ReLU导致的负值截断问题；解决了仿真中位姿对齐、坐标系转换等工程问题，实现了稳定抓取。

## 技术栈
- **深度学习**：PyTorch、VGG16
- **物理仿真**：MuJoCo 2.3.7、dm_control
- **机器人学**：ikpy、transformations
- **数值计算**：NumPy、OpenCV

## 项目成果
1.  完整复现了PoseCNN 6D姿态估计网络，实现了复杂场景下多物体的位姿精准预测；
2.  搭建了从视觉感知到机器人抓取的完整Pipeline，在MuJoCo中实现了稳定的无碰撞抓取；
3.  输出了详细的学习笔记，记录了网络原理与调试过程。

## 学习收获
掌握了6D物体姿态估计的核心原理，理解了深度学习视觉方法与机器人运动规划的结合方式，熟悉了具身智能视觉抓取任务的完整落地流程。
